# Dataset for the Similarity-Based Adaptive Goal Recognition System

This directory contains all datasets required to run both phases of the system.

---


# Dataset Contents
1. **domains.zip**

- Used in **Phase 1**
- Includes 13 IPC domains containing goal definitions and problem instances.

2. **experimental_data.zip**

- Used in **Phase 2**
- Contains observation traces (training/testing logs) for all domains and problem categories

---


# Usage Instructions

Extract both `.zip` files inside this directory.

Run the execution scripts described in:

codes/README.md
